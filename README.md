# ola-booking-data-analysis
End-to-end data analysis of Ola ride bookings using SQL, Power BI, and Excel to uncover booking trends, cancellations, revenue, and customer insights.
# 🚖 OLA Ride Booking Data Analytics Project

A complete end-to-end **Data Analytics** project that analyzes OLA ride booking data using **Python, SQL, and Power BI**. The project covers the complete analytics workflow—from data cleaning and exploratory data analysis (EDA) to SQL querying, interactive dashboard creation, report generation, and presentation. It demonstrates practical data analysis skills used in real-world business scenarios.

---

# 📌 Project Overview

This project focuses on analyzing ride booking data to uncover customer behavior, booking trends, cancellations, revenue insights, ride distances, and customer/driver ratings.

The workflow includes:

- Data Cleaning using Python
- Exploratory Data Analysis (EDA)
- SQL Analysis (PostgreSQL/MySQL/SQL Server)
- Interactive Power BI Dashboard
- Business Report
- Presentation (Gamma/PowerPoint)

---

# 📂 Dataset

The dataset contains **100,000 ride booking records** for Bengaluru city with information such as:

- Booking Date
- Booking Time
- Booking Status
- Customer ID
- Vehicle Type
- Pickup & Drop Locations
- Ride Distance
- Booking Value
- Payment Method
- Driver Ratings
- Customer Ratings
- Cancellation Details
- Incomplete Ride Details

The dataset is designed to simulate real-world ride booking operations.

# ▶️ How to Run

## Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/OLA-Data-Analytics-Project.git
```

## Step 2: Install Python Libraries

```bash
pip install pandas numpy matplotlib
```

## Step 3: Run the Python Analysis

```bash
python EDA.py
```

Or open the Jupyter Notebook.

## Step 4: Execute SQL Queries

- Import the dataset into PostgreSQL, MySQL, or SQL Server.
- Run the SQL script.
- Execute the analysis queries.

## Step 5: Open Power BI Dashboard

- Open `Ola.pbix`
- Refresh the dataset if required.
- Explore the dashboard using filters and slicers.
